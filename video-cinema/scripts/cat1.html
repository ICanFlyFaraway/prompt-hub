CLIENT_IP: "117.128.51.61"

Origin Request Timeout / Bad Gateway

What happened?

The CDN edge node successfully received the request, but the origin server failed to respond within the configured time limit.

The server, while acting as a gateway or proxy, did not receive a timely response from the upstream server.

How to solve?

As a visitor: The origin server may be slow or experiencing network congestion. Please try again later.

Administrator Troubleshooting Tips: 1. Check the origin server load and verify whether CPU or database performance is a bottleneck.
2. Check firewall rules on the origin server to ensure CDN nodes are not blocking long-lived connections.
3. Verify the network quality between the CDN and origin, or consider increasing the timeout settings on the CDN side.
